[2].이걸 1월~6월 까지 각 장르가 월별로 기록한 판매지수 총합을 모아 그래프로 그리기

//1번에서 구한 장르별 책들의 판매지수(6개월 누적 판매량) 총합 구하기
ex> 1월 베스트에 판타지 장르의 책이 20권이 올랐다면 그 20권의 누적 판매지수의 총합(= 장르의 월별 인기점수) 구하기

>>>> 이렇게 그래프로 표현하면 뭘 알 수 있지?

(1) 월별 장르 인기 변동 추이

: 각 장르들의 월별 인기점수 (= 판매지수 총합) 을 비교 -> 장르 인기도의 월별 변동 확인 가능
-> 장르의 인기도 변동이 별로 없다면 그 장르의 인기가 지속적으로 변동 없이 유지 된다는 것 확인 가능
-> but. 특정 달에 특정 장르의 인기점수가 갑자기 높아진다거나 감소한다면 계절적인 요인이나 영화개봉 같은 외부 요인들이 해당 장르의 판매에 영향을 미쳤을 가능성 존재

(2) 장르별 인기 순위와 월별 점수의 관계?

: 장르별로 계산한 [전체 판매지수 총합]와 월별 점수의 변동을 비교
-> 인기가 높은 장르가 실제로 매월 베스트 순위에서도 자주 등장하는지 확인 가능
-> 만약 인기 순위가 높은 장르가 대부분의 월에서 높은 점수를 유지한다면, 인기도가 베스트 순위와 연관성을 가지고 판매에 영향이 있다고 볼 수 있음
-> but. 인기도가 높은 장르가 특정 월에서는 베스트 순위에 적게 등장하거나 낮은 점수를 기록하면 소비자 취향 변화나 시즌 트렌드의 영향을 받았을 가능성이 있음

(3) 장르별 월별 판매 패턴 예측에 활용

: 이런 점수 변동을 가지고, 특정 장르의 월별 인기도 변화 예측 가능
-> 어떤 장르가 특정 계절에 인기가 상승한다면 그 패턴을 분석하면 다음 시즌에도 유사한 판매 패턴 예측 가능

[3] 장르별 월간 순의 등장 빈도 그래프로 만들기

 장르별 월간 등장 횟수 비교 : 매월 각 장르가 베스트 120위에 오른 횟수 기록 -> 월별 등장 횟수 변화 확인
 ex> x축 = 월(4~10), y축 = 베스트 순위 내 장르별 책들의 등장 횟수

[4] 장르 인기 총합과 월별 등장 빈도 비교

 장르 판매지수 총합과 월별 등장 빈도 비교 :  각 장르의 총 판매지수(인기도)와 월별 등장 횟수 비교
 -> 인기도가 높을수록 월간 베스트 순위에 자주 등장하는지 확인

[5] 장르의 인기도 점수를 기반으로 판매량 예측하기
